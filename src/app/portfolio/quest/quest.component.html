Questi

Questi is a social media application that seeks to deliver authentic and first person interaction between a user and his/followers.

Objective

Questi is a social media application that seeks to deliver authentic and first person interaction between a user and his/followers.

It works through a Q and A Format. Each user has the ability to ask a question and answer a question. Whoever asks the question remains anonymous forever. You can ask anyone anything without the fear of being noticed. It is raw information and a place where you can be yourself. When you ask a user a question only you will know that you asked the question and now one else. The user who received the question will not know that you have sent the question! This is the perfect way to ask someone personal questions that you might not be able to ask them face to face.
